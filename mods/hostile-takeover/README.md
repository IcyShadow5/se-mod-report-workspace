# Hostile Takeover

## Context

Subsystem: NPC grid takeover / block classification / AiEnabled and WeaponCore interaction.

## Purpose

Track Hostile Takeover findings, sanitized evidence, and outgoing reports if a real mod-side issue is confirmed.

## Current known distinction

Hostile Takeover being loaded and printing configuration/classification data is not automatically a bug. It must be separated from actual NPC spawn-provider failures.
