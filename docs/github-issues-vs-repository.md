# GitHub Issues vs Repository Uploads

## Repository files

Only the owner and collaborators can upload or change repository files.

## Issues

If Issues are enabled on a public repository, other GitHub users can usually create issues and comments. They still cannot commit files to the repository unless invited as collaborators.

## Templates

Issue templates only guide the form. They do not give people repository write access.

## Recommended use

For my own server notes: keep the repository private.

For reporting a bug to a mod author: copy one prepared report from `reports/outgoing/` into the mod author's GitHub issue tracker, Steam Workshop comments, or Discord support channel.
